this repo simrs-api is a main backend for simrs-electron, simrs-web-master and simsr-public-mobile.
So any typing used will most likely used in other repository.To manage typing we define shared repository that contain all shared typing, and we can import it to simrs-api, simrs-electron, simrs-web-master and simsr-public-mobile. The shared repository is located in `shared` folder in the root of the project.

if needed we can run 'npm run yalc' to update the package locally
